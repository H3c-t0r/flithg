flyteidl.admin.ProjectDomainAttributes¶

[flyteidl.admin.ProjectDomainAttributes proto]

{
  "project": "...",
  "domain": "...",
  "attributes": "{...}"
}
project

(string) Unique project id for which this set of attributes will be applied.

domain

(string) Unique domain id for which this set of attributes will be applied.

attributes

(map<string, string>) Custom resource attributes which will be applied in cluster resource creation (e.g. quotas). Map keys are the case-sensitive names of variables in templatized resource files. Map values should be the custom values which get substituted during resource creation.
