create_part(name=None)[source]¶

Method which will return a Blob object for writing into a multi-part blob.

Parameters

name (Text) – [optional] If name is provided, it is a specific partition name to place as part of the multipart blob. When we read blobs from a multipart, it will be read in lexicographic order so this can be used to enforce ordering. If not provided, the name is randomly generated.
